# Shaunobi Homebrew Tap

Install `tloc` directly from this tap:

```sh
brew install shaunobi/tap/tloc
```

Alternatively, add the tap first:

```sh
brew tap shaunobi/tap
brew install tloc
```

`tloc` counts source lines and LLM tokens in one pass. Project documentation
and release notes are available at <https://github.com/shaunobi/tloc>.
